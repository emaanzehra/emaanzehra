## 🩺 About Me

I'm an Artificial Intelligence student with 1+ year of hands-on experience shipping production-grade systems across **machine learning**, **computer vision**, and **full-stack development**.

My work sits at the intersection of intelligent software and meaningful impact — and my long-term goal is to bring AI into healthcare, where it can genuinely transform how the world heals.

I've shipped **three end-to-end systems**: an AI dental cavity detection tool using TensorFlow and computer vision, a live news intelligence SaaS platform powered by Gemini API and LLMs, and a real-time weather microservice in Spring Boot serving 200,000+ cities.

At **Google Developer Groups On Campus**, I led development across 6 Agile sprints, mentored 8 junior developers, ran ML and Generative AI workshops for 50+ participants, and automated CI/CD pipelines that cut deployment effort by 60% — all with zero production incidents.

Currently pursuing a **Duales Studium Informatik B.Sc.** in Germany (start 2026), combining university study with industry practice in software engineering and AI.

> *"Write clean code, test everything, ship nothing broken."*

## 🌸 Featured Projects

### 🦷 [DentalVision — AI Radiographic Cavity Analysis](https://github.com/emaanzehra/dentalvision)
> *The project that started my obsession with AI in healthcare.*

A desktop AI diagnostic tool for real-time dental cavity detection from X-rays.
- **5-stage radiographic pipeline**: grayscale → CLAHE contrast enhancement → gamma correction → TensorFlow inference → live annotation
- **SSD MobileNet V2** object detection at 0.30 confidence threshold, rendering bounding boxes at 30+ FPS
- **MVC-decoupled Tkinter GUI** with 4 panels, live status indicators, and breadcrumb tracker
- Zero runtime inference failures across 50+ test radiographs

`Python` `TensorFlow` `OpenCV` `SSD MobileNet V2` `Tkinter` `Computer Vision`

---

### 📰 [NexusBrief — AI SaaS News Intelligence Platform](https://github.com/emaanzehra/nexusbrief)
> *120+ stories a day. Zero manual effort. Full transparency.*

A multi-category SaaS platform powered by LLMs for real-time news monitoring, summarisation, and sentiment analysis.
- **Gemini API + prompt engineering** for NLP-driven text classification across 5+ categories
- **Secure REST API** with token-based authentication across 4+ endpoints, 0 unauthorised access incidents
- Scalable **PostgreSQL** cloud pipeline with real-time analytics dashboard

`Python` `Laravel` `Gemini API` `LLM` `PostgreSQL` `REST API` `NLP` `Cloud Deployment`

---

### 🌤️ [WeatherWise — Real-Time Microservice](https://github.com/emaanzehra/weatherwise)
> *200,000+ cities. Sub-500ms. Always up.*

A cloud-native Spring Boot microservice for live weather and AQI data.
- **Horizontal scalability** support with 3 RESTful endpoints
- Clean **OOP/MVC** with software design patterns, structured exception handling, and unit testing
- 100% data integrity under concurrent load testing, 0 critical defects

`Java` `Spring Boot` `REST API` `OOP/MVC` `Microservices` `Cloud Deployment` `Unit Testing`
